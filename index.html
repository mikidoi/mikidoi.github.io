Webb om 5 år

För 5 år sedan när jag började lära mig programmering, fanns det inte så många som pratade om Responsiv design eller mobil anpassade webbsidor. Vi kan säga att webben utvecklades mycket inom senaste 5 år. Troligen ska det hända stora ändringar om nästa 5 år också. De nya teknikerna som finns redan idag, till exempel Google glass och Apple smart watch, ska bli mer vanligare och det ska komma nya tekniker som man kan klä på sig. (Wearable computers) Örhängen, halsband och ring kan det vara nästa produkter som nya wearable computers. I början ska det vara inte så snygg men om de jobbar med produkt designer, blir det trendigt. När webben dyker upp till våra vardagar, såg de inte snygg ut. Alla tekniker kommer först och design för dem ska utvecklas lite senare. De tekniker som finns redan idag kommer få mer design och interaktiva funktioner.


Medan jag arbetar som frontend utvecklare, lär jag mig många verktyg som kan göra lättare att programmera. Till exempel, med grunt eller gulp, kan vi kompilera javascript filer. Med grunt-autoprefixer, behöver vi inte skriva css koden för alla olika browser. Det hämtar informationer från databas på caniuse.com och kolla Updates hela tiden och vi kan skriva bara en rad av css för alla browser . Det finns redan många verktyg som hjälper utvecklare att göra deras jobb snabbare och mer effektivt senaste tiden men om 5 år blir det ännu mer, tror jag.


Och det ska bli även lättare att hitta informationer på nätet. För vanliga människor och företag som vill hitta rätta kunder. Det blir ännu lättare att köpa något med mobil. Utan att vara irriterad av onödiga spam, kan vi få rätt inbjudan och rabatt meddelande. Det kan leda till hårdare konkurrens mellan företag men det blir bra för kunder.


Som frontend utvecklare finns det många saker som vi kan bidra till utveckling av webben om nästa 5 år. UX är väldigt viktigt. Hur vi kan ge roliga upplevelse till människor. Användarvänlighet. Att skapa bra upplevelse och känslor för att låta besökare att känna säkra att köpa på nätet, mobil och andra nya tekniker. Personligen är jag mycket intresserad av design och undrar om hur det ska hända med webb design också. Man brukar följa trend men samtidigt försöker vissa webb designer skapa sina egna unika design. Så samma som mode, kan retro design komma tillbaka i framtiden? Det låter väldigt intressant att fortsätta jobba som utvecklare!

Om Miki
